# Основные сведения

## Схема работы агентов

```
      course-methodist        course-researcher         course-writer
      ─────────────           ───────────────            ───────────
      Последовательность      Ищет доки через            Пишет готовый
      и полнота тем           search_engine +             текст урока
                              tree.md + web

          |                        │                         │
          ▼                        ▼                         ▼
   main.md                   docs-01-...md              N_M_Название.md
   (структура курса)         docs-02-...md              (готовый урок)
   structure.md              docs-внешние.md
   (темы урока)                   │                         │
          │                        │                         │
          └────────┬───────────────┘                         │
                   ▼                                         │
        course/structure/N/M/                                │
                   └─────────────────────────────────────────┘
```

## Два уровня структуры

| Уровень | Название | Файл |
|---|---|---|
| 1 | Раздел курса | `course/structure/main.md` (таблица) |
| 2 | Урок | `course/structure/{N}/{M}/structure.md` (список тем) |

## Инструменты

| Инструмент | Путь | Назначение |
|---|---|---|
| Поисковый движок | `scripts/search_engine.py` | FTS5+TF-IDF поиск по doc-файлам |
| Переиндексация | `python3 scripts/search_engine.py index` | Запустить после обновления документации |

## Последовательность работы

| Шаг | Skill | Результат |
|---|---|---|
| 1 | course-methodist | `course/structure/{N}/{M}/structure.md` (список тем урока) |
| 2 | course-researcher | `course/structure/{N}/{M}/docs-*.md` (материалы) |
| 3 | автор | правки и заметки в папке |
| 4 | course-writer | `course/chapters/{N}_{M}_Название.md` (готовый текст) |

## Структура файлов

    course - Корневая директория курса
        chapters - Папка с готовыми текстами уроков
            intro.md - Введение
            1_1_Название.md - Урок 1.1
            1_2_Название.md - Урок 1.2
        images - Папка скриншоты и картинки общие для курса
            menu - Папка картинки кнопок меню
        structure - Папка структуры курса
            main.md - Структура разделов и уроков
            1 - Папка Раздел 1
                1 - Папка Урок 1.1
                    structure.md - Темы урока
                    docs-*.md - Материалы исследователя
                2 - Папка Урок 1.2
                    structure.md
            2 - Папка Раздел 2
