
# Основные сведения

## Схема работы агентов

```
      course-methodist        course-researcher         course-writer                         
      ─────────────           ───────────────            ───────────                         
      Строит структуру        Ищет доки через            Пишет готовый                       
      главы                   search_engine +            текст главы                         
                              tree.md + web                                                   
          |                        │                         │                               
          ▼                        ▼                         ▼                               
   structure.md            docs-01-...md               ЧастьX_Глава.md                       
                           docs-02-...md                                                     
                           docs-внешние.md                                                    
          │                        │                         │                               
          └────────┬───────────────┘                         │                               
                   ▼                                         │                               
        book/structure/1/3/                                  │                               
                   └─────────────────────────────────────────┘                               

## Инструменты

| Инструмент | Путь | Назначение |
|---|---|---|
| Поисковый движок | `.pi/tools/search_engine.py` | FTS5+TF-IDF поиск по 2016 doc-файлам |
| Переиндексация | `python3 .pi/tools/search_engine.py index` | Запустить после обновления документации |
 ```

## Последовательность работы

| Шаг | Skill             | Результат                         |
| --- | ----------------- | --------------------------------- |
| 1   | course-methodist  | course/structure/1/3/structure.md |
| 2   | course-researcher | course/structure/1/3/docs-*.md    |
| 3   | автор             | правки и заметки в папке          |
| 4   | course-writer     | course/chapters/1_3_Название.md   |

## Структура файлов

    course - Корневая директоря книги
        chapters - Папка с главами
            1 - Папка Часть 1
                images - Папка скриншоты для этой части
                1_название.md - Файл Глава 1
            2 - Папка Часть 2
        images - Папка скриншоты и картинки общие для книги
            menu - Папка картинки кнопок меню
        Введение.md - Файлы общие книги
